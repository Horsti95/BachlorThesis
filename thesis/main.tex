\begin{abstract}
Standard ML experimentation pipelines face two systematic failure modes during iterative development: \textit{incorrect orchestration}, where operations such as feature selection are applied globally before cross-validation splits, risking subtle data leakage; and \textit{redundant computation}, where unchanged experiment configurations trigger full retraining across every cross-validation fold. In Leave-One-Subject-Out (LOSO) cross-validation with 128 subjects and 18 parameter configurations, uncached retraining can exceed 21 hours per iteration. This thesis addresses both failure modes through a fingerprint-based caching framework utilizing SHA-256 configuration hashing.

The research develops a deterministic fingerprinting system where a cryptographic hash uniquely identifies each experiment configuration. The fingerprint is derived from a canonical JSON representation of the tracked experiment state, including the random seed, code version, model hyperparameters, feature configuration, and held-out subject. This ensures that any modification to parameters that may affect the output results in a unique fingerprint, enabling automatic cache invalidation. Consequently, identical fingerprints indicate an identical tracked configuration, allowing for valid, reproducible cache retrieval.

The framework is integrated into a multi-stage ML pipeline and specifically extends this mechanism to Leave-One-Subject-Out (LOSO) cross-validation. By incorporating the held-out subject into the cache key, the system ensures cache integrity and prevents the improper reuse of models across different folds, thereby maintaining strict isolation between training and testing sets under a fixed dataset partition.

Validation is conducted via a sleep stage classification case study using the Bitbrain Open Access Sleep (BOAS) dataset (128 subjects, approximately 120,000 epochs). The feature extraction module computes 149 features per epoch across six EEG channels, which are cached persistently to isolate the performance evaluation to the computationally intensive model training phase. Feature selection (ANOVA F-statistic) is applied globally across all subjects prior to LOSO splits---a deliberate design choice that enables consistent cache fingerprinting at the cost of negligible label overlap ($<$1\%).

The contributions of this thesis are: (1) a fingerprint-based LOSO model caching framework that ensures cache correctness and prevents cross-fold contamination; (2) quantified efficiency gains, demonstrating a median speedup of 54$\times$ for XGBoost and 11.6$\times$ for Random Forest across 128 LOSO folds; and (3) validated reproducibility, where all cached models yielded identical classification results to those obtained through cold retraining.
\end{abstract}
