\begin{abstract}
Sleep stage classification is a critical task in sleep medicine and neuroscience research. This thesis presents an automated sleep stage classification pipeline using machine learning models trained on EEG data from the BOAS dataset. We implement and compare two models---XGBoost and Random Forest---using Leave-One-Subject-Out (LOSO) cross-validation to ensure subject-independent generalization.

A key contribution of this work is the implementation of a two-tier caching system consisting of feature-level and model-level caches with SHA-256 fingerprint-based cache invalidation. This approach significantly reduces computational overhead for iterative experimentation while maintaining full reproducibility.

Our pipeline extracts 149 time-domain and frequency-domain features from 6 EEG channels per 30-second epoch, applies ANOVA-based feature selection, and trains models on 128 subjects (approximately 119,763 epochs total). The LOSO cross-validation ensures robust evaluation of subject-independent performance.
