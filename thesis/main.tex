\begin{abstract}
This thesis addresses the computational costs and reproducibility challenges in modern Machine Learning (ML) experimentation, where redundant computations and inconsistent results create significant overhead. To mitigate these issues, this work proposes a deterministic caching framework utilizing SHA-256 configuration hashing.

By deriving a cryptographic fingerprint from a canonical JSON representation of the experiment state, including random seeds, code versions, hyperparameters, and held-out subjects—the system ensures that any parameter modification results in automatic cache invalidation. This mechanism specifically supports Leave-One-Subject-Out (LOSO) cross-validation by incorporating the subject identifier into the cache key, preventing cross-fold contamination and maintaining strict isolation between training and testing sets.

The framework was validated via a sleep stage classification case study using the Bitbrain Open Access Sleep (BOAS) dataset (128 subjects, ~120,000 epochs). The feature extraction module computes 149 features per epoch, which are cached to isolate performance evaluation to the model training phase.

The contributions are: (1) a validated LOSO model caching framework ensuring cache correctness; (2) quantified efficiency gains, with median speedups of 178$\times$ for XGBoost and 17.7$\times$ for Random Forest; and (3) verified reproducibility, where cached models yielded results identical to cold retraining.
\end{abstract}
